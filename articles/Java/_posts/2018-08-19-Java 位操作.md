## Java 中的三种移位运算符
Java 包含`<<`、`>>`、`>>>`三种移位运算符
```java
int num = 8;
int offset = 3;
System.out.println(Integer.toBinaryString(num));

// 左移: num = num * 2^offset
num = num << offset;
System.out.println(Integer.toBinaryString(num));

// 右移: num = num / 2^offset
num = num >> offset;
System.out.println(Integer.toBinaryString(num));

// 无符号右移: num = num / 2^offset，最高位补零
num = num >>> offset;
System.out.println(Integer.toBinaryString(num));
```
